# WCMSPolls
A Laravel package to manage polls on Darkpony Barb CMS


## Installation:
First, install the package through Composer. 

```bash
composer require darkperis/wcms-polls
```

Publish migrations, and migrate

```bash
php artisan vendor:publish
php artisan migrate
```
